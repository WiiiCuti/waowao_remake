**Non-Human Characters** (animals, mythical creatures, well-known figures, etc.):

【Visual Tier Specifications】

⚠️ Core Principle: Costume ornateness is determined by the character's social identity and plot setting, NOT by importance tier!

S-Tier Characters:
  - Description length 300-500 characters
  - Must have extremely high visual distinctiveness and "protagonist aura"
  - Clothing style determined by character identity (a poor student can wear a simple school uniform, but facial features and temperament must be outstanding, with striking appearance)
  - Include fabric texture, embroidery/jewelry details, hair accessories, every distinctive element

A-Tier Characters:
  - Description length 250-400 characters
  - Must have distinct personal characteristics and memorable points
  - Include clothing details, accessories, hairstyle specifics
  - Clothing style determined by character identity

B-Tier Characters:
  - Description length 180-300 characters
  - Must have basic identifying features with sufficient visual detail
  - Clothing style matches their social identity

C-Tier Characters:
  - Description length 120-200 characters
  - Concise but complete image description with key visual elements

D-Tier Characters:
  - Description length 80-150 characters
  - Basic image covering gender, face, hair, and visible clothing

【Costume Ornateness Tier Guide】
Tier 1 (Plain/Uniform) → Tier 2 (Everyday/Ordinary) → Tier 3 (Professional) → Tier 4 (Elite/Aristocracy) → Tier 5 (Royalty/Ultimate Luxury)

⚠️ **Facial Archetype System (Critical!)**

Select ONE facial archetype based on character personality, profile, and story genre. Each archetype defines the character's facial style — this prevents same-face syndrome and ensures each character has a distinct look fitting their story context:

1. **Ordinary Realism** (Đời thường / Bình dị) → For Slice-of-Life, realistic, or everyday characters
   - Key features: Natural unassuming features, gentle everyday look, approachable plain beauty, soft realistic facial structure
   - Vibe: Warm, grounded, realistic, unpretentious
   - Keywords: natural unassuming features, soft everyday facial contours, gentle plain beauty, realistic proportions

2. **Cool Beauty** (Lạnh lùng / Sắc sảo) → Aloof, cold, tsundere personalities (Urban/Romance/Drama)
   - Key features: Sharp slanted eyes (phoenix eyes), high cheekbones, thin lips, defined jawline, angular face shape
   - Vibe: Sharp, distant, striking, elegant severity
   - Keywords: slanted phoenix eyes, sharp defined jawline, high sculpted cheekbones, thin lips with sharp cupid's bow

3. **Sweet Beauty** (Dễ thương / Ấm áp) → Gentle, warm, lively personalities (Romance/Drama)
   - Key features: Large round or almond eyes, soft full lips, curved brows, soft oval or heart-shaped face
   - Vibe: Warm, approachable, innocent, bright
   - Keywords: large sparkling almond eyes, soft full lips, heart-shaped face, delicate rounded features

4. **Weathered/Gritty** (Từng trải / Phong trần) → Thriller, mystery, action, or survivor characters
   - Key features: Weathered skin, sharp tired eyes, rugged jawline, subtle signs of fatigue or experience
   - Vibe: Intense, experienced, hardened, survivalist
   - Keywords: weathered textured skin, sharp tired eyes, rugged jawline, subtle signs of fatigue

5. **Ethereal/Immortal** (Thần tiên / Siêu phàm) → Fantasy, cultivation, xianxia characters
   - Key features: Flawless otherworldly skin, striking unearthly symmetry, mesmerizing gaze
   - Vibe: Divine, untouchable, majestic, transcendent
   - Keywords: flawless otherworldly skin, striking unearthly symmetry, mesmerizing transcendent gaze

6. **Classic Beauty** (Thanh tú / Hài hòa) → Neutral, balanced personalities (default for general fiction)
   - Key features: Well-proportioned symmetrical features, moderate eyes, medium brows, balanced lips
   - Vibe: Elegant simplicity, timeless beauty, clean refinement
   - Keywords: perfectly symmetrical features, clear almond-shaped eyes, well-proportioned nose and lips, elegant refined contours

Applicable tiers:
- S-Tier & A-Tier: Must use an archetype that perfectly matches their narrative role. For realistic characters, use Ordinary Realism and describe them as authentically everyday (do not force 'striking' features). For dramatic/fantasy characters, features should be more exceptional.

【Sub-Appearance Rules】

Generate corresponding appearance descriptions based on the input expected_appearances:

Main appearance (id=0) must be a complete description, including:
- All basic features (face, eyes, hair, body type, etc.)
- Complete description of initial clothing/accessories
- Shoes are required

Sub-appearances (id>=1) only describe the visual change parts, since they will be based on modifying the main appearance image:
- Begin with gender + brief face reference: "Same Male. Same face: [hair color, face shape]. Changed: [new clothing, shoes]"
- This ensures sub-appearance descriptions are still usable even without the main image reference
- The face/hair summary must be <20 words — it's a reminder, not a full re-description
- Outfit change: Write only new clothing, shoes, and changed hairstyle if any
- Age change: Write appearance differences (wrinkles, white hair, etc.) + same face reference
- Special state: Post-bath, battle attire, etc. + same face reference
- Prohibited: Repeating full descriptions of basic face/body type features

Examples:
- Main appearance (id=0): "Female, early twenties, soft oval face with slim chin, large dark brown almond eyes with subtle eyeliner, refined straight nose, light pink lips with natural blush. Voluminous dark brown hair styled in elegant half-updo, long side braid draped over right shoulder adorned with small white pearl beads, loose wavy tendrils framing the face. Slender graceful build. Wearing off-white cream Hanfu with mandarin collar and traditional frog button closures, delicate cherry blossom embroidery in soft pink tones on chest and cuffs, loose mid-length sleeves. White silk inner robe visible at collar. Matching pearl drop earrings, white embroidered cloth flats."

- Battle attire (id=2): "Same Female. Same face: dark brown hair, almond eyes. Changed: braided hair pulled into tight high ponytail, dark fitted battle tunic with leather shoulder guards, dark trousers, black combat boots."
