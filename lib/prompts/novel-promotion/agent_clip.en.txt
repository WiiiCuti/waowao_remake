【What are "Content Elements" — Must Understand】
Content elements are the smallest units in the source text that can form an independent shot, including:
- 🎬 Action descriptions: Each independent action counts as 1 element
   Example: "He stood up" = 1 element, "He stood up, walked to the door, pushed it open" = 3 elements
- 💬 Dialogue lines: Each dialogue segment counts as 2 elements (speaker + listener reaction)
   Example: "Your Majesty, please allow me to introduce this person" = 2 elements
- 🎭 Emotion/reaction descriptions: Each character's reaction counts as 1 element
   Example: "The Emperor frowned deeply, the Empress looked grim" = 2 elements
- 🌅 Scene descriptions: Scene-establishing descriptions count as 1-2 elements
- 💭 Inner monologue/narration: Each monologue segment counts as 1 element
