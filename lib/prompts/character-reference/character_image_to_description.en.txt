# Image-to-Character-Description Prompt

Analyze this character image and generate a detailed character visual description (for AI image generation).

## Output Requirements

Generate a complete character visual description that includes the following elements:

1. Gender and approximate age (e.g. a male of about twenty-five years old)
2. Hairstyle and hair color (e.g. short black hair, slightly wavy long brown hair)
3. Face shape and facial features (e.g. sword-like eyebrows with bright eyes, high nose bridge, thin lips)
4. Body build and figure (e.g. slender build, robust physique)
5. Clothing style (e.g. dark blue suit, white shirt, leather belt)
6. Accessories and distinctive details (e.g. silver watch on the left wrist, golden brooch on the chest)
7. Overall temperament keywords (e.g. elite aura, restrained-and-cold type, aloof, gentle warm-hearted man)

## Missing Content Completion Rules

If the reference image only shows part of the body (e.g. upper body, headshot), reasonably infer and complete based on the available information:
- **Missing lower body**: infer pants/skirt type based on the top style (e.g. suit jacket paired with dark blue trousers, casual top paired with jeans)
- **Missing shoes**: infer shoe type based on the overall outfit style (e.g. formal wear paired with leather shoes, casual wear paired with sneakers or canvas shoes)
- **Missing accessory details**: reasonably add accessories based on the character's temperament (e.g. business style paired with a watch, casual style paired with a bracelet)

## Forbidden Descriptions

- Skin color
- Eye color
- Expression
- Action
- Background
- Pose

## Output Format

One coherent descriptive paragraph, about 200-300 words, directly usable as an image generation prompt.
Return only the description text; do not include any titles, numbering, or other formatting.
